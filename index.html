References

ChatGPT (OpenAI)

Free (limits) Plus / Team / Enterprise

Consumer access to OpenAI’s flagship models in the chat app. Latest default is GPT-5 (rollout varies by plan/region); GPT-4o remains a fast multimodal workhorse. GPT-5 · GPT-4o · GPT-5 in ChatGPT · Plus plan

  • Free tier: usage caps; newest model availability may be limited or variable by region/plan.
  • Paid: higher limits, priority features; Team/Enterprise add collaboration and admin controls. Pro plan
  • HPE fit: rubric-guided feedback, case prompts, portfolio review, image/code tasks.

OpenAI GPT-4.1 (API)

Developer (paid)

API series with major improvements to instruction following and **long-context (up to ~1M tokens)** — useful for large syllabi, policy corpora, and assessment banks. Overview · Docs · API pricing

Claude (Anthropic)

Free (usage-limited) Pro / Max / Team

General-purpose assistant with strong reasoning and writing. Latest families include Claude 4 and Claude 3.5 Sonnet. Claude 4 · 3.5 Sonnet · Plans & pricing

  • Free vs paid: primarily differs in message limits, newest model access, and features like web integration (varies by region and plan).
  • HPE fit: structured feedback, policy drafting, critique with citations.

Gemini (Google)

Free (Gemini app) Pro / Ultra

Multimodal assistant in apps; API offers long-context (up to ~1M tokens on 1.5/2.0 series) and advanced features. Gemini 2.0 · API models · Long context · Plans

  • Free vs paid: Pro/Ultra unlock higher limits, priority access, and new features; API has free and paid tiers.
  • HPE fit: multimodal lesson assets, rubric-guided summaries, large-document reviews.

How to choose (educator quick guide)

  1. Start with availability: design activities that work within the **free** plans most faculty and learners already have; add “Plus/Pro/Pro/Ultra” enhancements as optional.
  2. For large corpora: consider **OpenAI GPT-4.1 API** or **Gemini API** long-context features for whole-course materials and policy banks.
  3. Writing and critique: Claude paid tiers are strong for structured drafting and review.
